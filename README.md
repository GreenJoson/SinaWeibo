# SinaWeibo
>1.python脚本,支持非官方API 方式登录新浪微博发送消息和图片

>2.python2.7版在[https://github.com/XJouYi/SinaWeibo/releases/tag/v0.1]

## API
| 名称        | 参数    |  说明  |
| --------   | -----:  | :----: |
| uploadPic  | picPath                |   上传图片到微博       |
| postMessage| message                |   发送文本微博         |
| postImage  | message,*picPaths      |   发送文本加图片微博(图片可多张)    |
